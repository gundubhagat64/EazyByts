# 🌐 Personal Portfolio Website

A modern, responsive, and interactive developer portfolio built to showcase my skills, projects, education, and achievements. The portfolio provides a clean user experience with smooth animations and an easy way for recruiters and clients to learn more about me.

---

## ✨ Features

- 🏠 Responsive Home Page
- 👨‍💻 About Me Section
- 🛠️ Technical Skills
- 🚀 Projects Showcase
- 📄 Resume Download
- 📞 Contact Form
- 🌙 Modern UI Design
- ⚡ Smooth Animations
- 📱 Mobile-Friendly Layout

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/gundubhagat64/portfolio.git
```

### Navigate to Project

```bash
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
